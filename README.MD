# Anzen
Anzen is a script for securing a Virtual Machine image. It is intended for CyberPatriot competitions.

# Supported OSs
- Ubuntu 22.04 LTS
- Ubuntu 20.04 LTS
## Partially Supported
- Ubuntu 18.04 LTS
- Future Ubuntu Releases
## Unsupported
- Ubuntu 16.04 & Older
- Non-ubuntu Distros
